# Css 가상선택자

## 반응,상태,구조 선택자

상태를 지정하는 가상 선택자

```css
ex) 반응선택자
a:link {
    a태그에 링크되어 있는 웹사이트 방문전 상태를 표현
}
a:visited{
    방문후에 상태를 표현
}
    a:active{
        마우스로 클릭한 상태를 표현
    }
a:hover{
    마우스 커서를 올려놨을때 상태를 표현
}

    상태선택자
input:focus{
    해당태그가 포커스를 가졌을때를 표현 (인풋박스에 텍스트를 입력 가능한상태등)
    백그라운드나,아웃라인 조정등 자주쓰임
}

구조선택자

:first-child{
    첫번째 자식요소를 선택할때 사용
}
:last-child{
 마지막 자식요소 태그를 선택
}
:nth-child(n){
    n번째 태그 요소 선택
}
:nth-last-child(n){
    끝에서부터 n번째 태그 요소 선택
}
li:nth-child(1){
    리스트 태그중 이와 동일 레벨에 있는 태그들 중에 첫번째 요소를 선택
}
li:nth-last-child(2){
    리스트 태그중 이와 동일 레벨에 있는 태그들 중에 마지막에서 2번째 요소를 선택
}
li:nth-last-child(1){
    리스트 태그중 이와 동일 레벨에 있는 태그들 중에 마지막에서 2번째 요소를 선택
}
```
